J1939 Verify DTC Page

J1939 Verify DTC contains the channels specific to the Verify DTC command.

Verify DTC Channels

# of DTCs Found

This channel returns the number of active and previously active DTCs for the device addressed through the Address to Query channel.

Expected DTC

This channel is used to tell the diagnostic command engine which DTC to look for. Set this channel to the desired value before setting the Verify DTC command.

Requested DTC Status

This channel returns the result of the Verify DTC command. Values are as follows:

0 - DTC Not Present/Not Found

1 - DTC Active

2 - DTC Previously Active
