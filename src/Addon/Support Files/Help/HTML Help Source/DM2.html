J1939 DM2 Active DTCs Page

J1939 DM2 Active DTCs contains sections for configuring DM2 (previously active) DTCs.

What do you want to do?

Add DTCs

Right click on DM2 Previous DTCs and select Add DTC.

Delete Multiple DTCs

To delete multiple items, right-click desired Diagnostics in the configuration tree and select Delete Multiple DTCs from the shortcut menu.
