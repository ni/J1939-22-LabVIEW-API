Counter Configuration

The Counter signal for an outgoing frame contains the rolling counter value.

What do you want to do?

Set Rolling Counter Max value

To set a Max value for the signal, enter the desired value in the Rolling Counter Max Count field. Note that the Max Count is inclusive, e.g. when this value is 15, the counter will increment 0, 1, 2....15 and then restart at 0.

Deselect signal as Counter signal

To set the signal back to a non-counter signal, right-click on the signal and select Make Signal.
