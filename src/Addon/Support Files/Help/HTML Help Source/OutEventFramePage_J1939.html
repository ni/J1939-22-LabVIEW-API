J1939 Outgoing Event Triggered Frame Page

J1939 Outgoing Event Triggered Frame contains J1939 parameters from the highlighted frame.

What do you want to do?

Set Disable Transmission Channel

The Disable Transmission channel will stop all CAN generation for the given frame when the channel is not set to 0. When the channel is 0, transmission will occur as expected. To set the Disable Transmission channel, select Disable Transmission and then browse to the desired channel.

Set Transmission Trigger

The Transmission Trigger allows various options for sending event frames.

  • Channel value change - Triggers when any signal contained within the frame has a value change.
  • Trigger channel (not zero) - Triggers when the Trigger channel has a value change to something that is not zero.
  • Channel value change or trigger channel (not zero) - Triggers when either of the previous conditions is true.
  • Trigger channel (any value) - Triggers when the Trigger channel has a value change.
  • Channel value change or trigger channel (any value) - Triggers when either of the previous conditions is true.
Select the desired trigger option from Transmit Trigger and then browse to the desired channel (if appropriate).

Delete

To delete a frame, right-click the desired frame in the configuration tree and select Delete from the shortcut menu.

Delete multiple parameters

To delete multiple parameters, right-click the desired frame in the configuration tree and select Delete Multiple Parameters from the shortcut menu.
