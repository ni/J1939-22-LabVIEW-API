J1939 Device Page

J1939 Device contains name and address parameters for the simulated node.

What do you want to do?

Rename Device

To rename the device, enter the desired text into the Name field.

Set J1939 Address

Set the J1939 network address for the custom device in the J1939 Network Address field. This is the address the custom device will claim on the network. Select an address value that is not claimed on the network as this custom device does not neogiate if another device exists at the requested address.

Set J1939 Name

Set the J1939 network name for the custom device in the J1939 Name field.

Delete

To delete, right-click the desired item in the configuration tree and select Delete from the shortcut menu.

Delete multiple

To delete multiple items, right-click desired device in the configuration tree and select Delete Multiple from the shortcut menu.
