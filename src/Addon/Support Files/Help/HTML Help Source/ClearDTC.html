J1939 Clear DTCs Page

J1939 Clear DTCs contains the channels specific to the Clear DTCs command.

Clear DTCs Channels

DTC Clear Type

This channel sets the type of DTCs to clear for the device addressed through the Address to Query channel. Set this channel to the desired value before setting the Verify DTC command. The values are as follows:

0 - Clear Active and Previously Active DTCs

1 - Clear Active DTCs

2 - Clear Previously Active DTCs
