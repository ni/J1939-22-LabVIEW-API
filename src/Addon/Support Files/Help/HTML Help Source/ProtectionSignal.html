Protection Configuration

The Configuration signal for an outgoing frame contains the protection value.

What do you want to do?

Select Channel Configuration

To select the channel configuration, right-click on the protection signal and select Select Channels. Select the data signals to be used in computing the protection value. All required information for the data signals will be pulled from the database.

Select Protection Type

To select the protection type, click on Protection Type and select appropriate option from the drop-down menu.

Deselect signal as Configuration signal

To set the signal back to a non-protection signal, right-click on the signal and select Make Signal.

Delete

To delete an item, right-click the object in the configuration tree and select Delete from the shortcut menu.
