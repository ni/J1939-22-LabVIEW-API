J1939 Device Simulation Page

J1939 Device Simulation allows for simulation of multiple node addresses. Adding a device to this section will get the addon to claim and protect the address on the network.

What do you want to do?

Add Device

To add a device to simulate, right-click Simulation and select Add Devicein the configuration tree from the shortcut menu.

Delete multiple

To delete multiple items, right-click on Simulate in the configuration tree and select Delete Multiple from the shortcut menu.
