J1939 DTC Frame Page

J1939 DTC contains J1939 DTC parameters from the highlighted frame.

What do you want to do?

Enable DTC

To enable the DTC, set the Active channel to 1 when the system is running. This will cause the addon to include the DTC in any response to a request for DTCs.

Delete

To delete a frame, right-click the desired frame in the configuration tree and select Delete from the shortcut menu.
